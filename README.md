# SMILE_BASIC
SMILE_BASICとはSmileBoom社が開発したプログラミング言語です。3DSやSwitchで当該言語を用いてプログラミングが出来るようになっています。3DSではプチコン3号、Switchではプチコン4というソフトウェアが対応しています。
そして、ここはプチコン厳選自作作品集です。コードにつきましては、プチコン３号の作品はパソコンへコードを出力できたのですが、  
プチコン4の作品は出力できませんでした。
また、コードには文字化けがあります。
